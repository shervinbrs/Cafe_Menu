## General Information

A cafe menu system with admin panel
make your own cafe menu with advanced detail


## Features

- Menu Full Manage
- Category Full Manage
- Item Full Manage
- General Setting
- Authentication

## Technologies Used

- Laravel Framework 10.24.0

